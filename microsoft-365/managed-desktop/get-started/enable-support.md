# <a name="enable-user-support-features"></a>Kullanıcı desteği özelliklerini etkinleştirme

İster kendi kullanıcı desteğinizi sağlarken ister destek sağlamak için bir iş ortağıyla çalışıyor olun, destek sağlayıcısının yükseltilmiş cihaz erişimi istemesine veya gerekirse sorunları Microsoft Yönetilen Masaüstü'ne yükseltmesine olanak sağlamak için aşağıdaki adımları izleyin.

1. Henüz hesabı yoksa, kullanıcıların Microsoft Yönetilen Masaüstü cihazlarıyla aynı Azure Active Directory (AAD) etki alanında bir hesaba ihtiyacı vardır.
1. Kullanıcı hesaplarını çalışma alanı içinde **Modern workplace Roles-Support İş** Ortağı güvenlik grubuna ekleyin Azure Active Directory (AAD).
