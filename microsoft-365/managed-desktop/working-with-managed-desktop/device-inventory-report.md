- Seri Numarası
- Cihaz Adı
- Cihaz Son iade
- Yönetilen
- Üretici
- Model
- işletim sistemi
- Işletim Sistemi Lisansı
- Işletim Sistemi Sürümü
- Sahiplik
- Kayıt Tarihi
- Halka Adı
- Depolama - GB ücretsiz
- Depolama - GB toplam
- User UPN tarafından kayıtlı
- Atanan Kullanıcı
- Durum
- Yaş (Ay)
- Profil
- Oturum Açan Kullanıcılar
- TPM sürümü
- Güvenli Önyükleme Etkin
- Birincil Disk Türü
- Toplam Fiziksel Bellek
