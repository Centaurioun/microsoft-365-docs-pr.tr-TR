# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>SharePoint Server 2013 kullanarak Microsoft Azure Internet Siteleri

 Azure Altyapı Hizmetleri'SharePoint tarafından barındırılan İnternet siteleri SharePoint Server 2013 avantajından yararlanabilir. Bu makale, bu çözümü tasarlama ve uygulamaya için kaynaklar sağlar.

## <a name="using-azure-infrastructure-services-for-internet-sites"></a>İnternet siteleri için Azure Altyapı Hizmetleri'nin kullanımı

Microsoft Azure, SharePoint Server 2013'ü temel alan İnternet sitelerini barındırmak için cazip bir seçenek sağlar. Avantajlar şunlardır:

- Altyapı geliştirmek yerine mükemmel bir site geliştirmeye odaklanın.

- Ölçeklendirerek ve ölçeklendirerek çözümlerinizi isteğe göre ölçeklendirme esnekliği.

- Yalnızca ihtiyacınız olan ve kullanabileceğiniz kaynaklar için ödeme ödenin.

- Müşteri hesapları için Azure Active Directory bu hizmetlerden faydalanabilirsiniz.

- Derin raporlama ve çözümleme gibi Microsoft 365 anda kullanılabilir durumda olan özellikleri ekleyin.
