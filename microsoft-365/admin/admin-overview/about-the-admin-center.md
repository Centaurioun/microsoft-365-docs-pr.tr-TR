# <a name="about-the-microsoft-365-admin-center"></a>Microsoft 365 yönetim merkezi hakkında

**İş planınız için Microsoft 365 ve yönetici izinlerine sahipseniz, yönetim merkezine erişebilirsiniz**. Bu makale tam size göre!

**Bir kullanıcı iseniz veya bir Microsoft 365 Aile planınız varsa, yönetim merkeziniz yok.** E-Microsoft 365 [kurmak için PC veya Mac bilgisayara Microsoft 365 ya da Office 2019'u](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658) indirme ve yükleme veya yeniden yükleme'ye gidin.

Aşağıdaki <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 yönetim merkezi bulutta ayarlamak</a>, kullanıcıları yönetmek, abonelikleri yönetmek ve çok daha fazlası için kullanabilirsiniz. Bu makalede, yönetim merkezine ulaşmayı öğrenecek ve kullanılabilir özellikler ve ayarlar hakkında bilgi edineceksiniz.

Yönetim merkeziyle ilgili kısa videoyu izleyin. <br><br>

## <a name="how-to-get-to-the-admin-center"></a>Yönetim merkezine gitme

1. Yönetici hesabınızla <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">admin.microsoft.com</a> adresinde oturum açın.

2. Sol üst köşedeki uygulama başlatıcı simgesini seçin ve ardından **Yönetici**’yi seçin.

    Yönetici **kutucuğu** yalnızca yönetici izinlerine sahip Microsoft 365 [görünür](../add-users/about-admin-roles.md). Kutucuğu göremiyorsanız, kuruluşunuzun yönetim merkezine erişim için gereken izinlere sahip değilsiniz demektir.
