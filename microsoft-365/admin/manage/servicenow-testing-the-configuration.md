# <a name="testing-the-servicenow-configuration"></a>ServiceNow yapılandırmasını test etme

Application requires successful communication with external systems, outline how to test the connection to ensure a successful configuration.

Destek tümleştirmesi için yapılandırmasını test etmek Microsoft 365 izleyin:

1. Yönetici olarak ServiceNow portalında oturum açın.

2. Herhangi bir olayı açın.

3. Önerilen çözümlerin **Microsoft 365 döndürül olup** olmadığını **belirlemek için Microsoft 365 Analizler** Destek sekmesine odaklanın ve Önerilen Seçenekler'i seçin.
