# <a name="testing-the-servicenow-configuration"></a>ServiceNow yapılandırmasını test etme

Uygulamanız dış sistemlerle başarılı bir iletişim gerektiriyorsa, başarılı bir yapılandırma sağlamak için bağlantının nasıl test edilmesinin ana hatlarını belirtin.

Microsoft 365 destek tümleştirmesinin yapılandırmasını test etmek için şu adımları izleyin:

1. ServiceNow portalınızda yönetici olarak oturum açın.

2. Herhangi bir olayı açın.

3. Önerilen çözümlerin başarıyla döndürülip döndürülmediğini belirlemek için **Microsoft 365 Destek** sekmesine odaklanın ve **Microsoft 365 İçgörüleri'ni** seçin.
