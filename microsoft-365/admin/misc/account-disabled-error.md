# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Web üzerinde Outlook'de hesap devre dışı hatası alma

Web üzerinde Outlook (eski adıyla `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Outlook Web App) açmaya çalıştığınızda **Hesabınız devre dışı bırakıldı** hatasını alırsanız yöneticiniz Web üzerinde Outlook erişiminizi devre dışı bırakmış olabilir.

Yöneticiniz, [posta kutusu için Outlook Web App etkinleştirme veya devre dışı bırakma](/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app) başlığındaki adımları izleyerek bu hatayı düzeltebilir.
