# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>E-postada hesap devre dışı Web üzerinde Outlook

Web üzerinde Outlook'i (eski  adı Outlook Web App) açmaya çalışsanız da Hesabınız devre dışı bırakıldı hatasını alırsanız, yöneticiniz bu hataya erişiminizi Web üzerinde Outlook.`X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException`

Yöneticiniz bu hatayı, Posta kutusunda posta kutusu için posta kutusunu etkinleştirme veya devre dışı [Outlook Web App adımlarını gerçekleştirerek düzeltebilir](/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app).
