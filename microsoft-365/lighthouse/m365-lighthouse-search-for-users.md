# <a name="search-for-users"></a>Kullanıcı arama

Microsoft 365 Lighthouse, tüm etkin kiracılarda belirli bir kullanıcıyı aramanızı sağlar. Bu sayede, farklı kiracılarda gezinmek zorunda kalmadan bir kullanıcıyı hızla bulabilirsiniz.

1. Deniz Feneri'nin sol gezinti bölmesinde Kullanıcılar'ı **seçin**.

2. Kullanıcıları ara **sekmesini** seçin.

3. Arama kutusuna bir görünen ad veya kullanıcı asıl adı (UPN) girin.

4. Arama sonuçları listesinde, araştırma yapmak istediğiniz kullanıcıyı seçin.

## <a name="next-steps"></a>Sonraki adımlar

Kullanıcı bulunduktan sonra parolasını sıfırlayabilirsiniz veya hesabını engelleyebilirsiniz. Daha fazla bilgi için bkz [. Kullanıcı parolasını sıfırlama](m365-lighthouse-reset-user-password.md) [veya Kullanıcı oturum açmasını engelleme](m365-lighthouse-block-user-signin.md).
