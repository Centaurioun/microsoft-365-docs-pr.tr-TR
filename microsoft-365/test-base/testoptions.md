# <a name="step-5-choose-your-test-options"></a>5. Adım: Test seçeneklerinizi seçin. 

Bu ```Test Options``` sekme, işlevsel testlerini gerçekleştirmek isteyen kullanıcılara, Windows Güncelleştirme düzeltme eklerinin, işlevsel test betiklerini yürütme sırasına göre uygulanması gerektiğini belirtmektedir.

![Test seçeneklerinin resmi. Her ikisinde de ilk sınama veya işlevsel testler.](Media/testoptions.png)

Sonraki _**sekmeye**_ gitmek ve seçili test seçeneklerinizi gözden geçirmek için Gözden Geçir'i seçin.
