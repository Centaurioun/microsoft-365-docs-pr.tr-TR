# <a name="whats-new-in-microsoft-secure-score"></a>Microsoft Güvenli Puanı'daki bu yenilik

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Microsoft Secure Score'u güvenlik nedenlerinizi daha iyi bir temsil etmek için bazı değişiklikler yaptık. Planlanan değişiklikler hakkında bilgi edinmek için bkz [. Microsoft Güvenli Puan'da neler var?](microsoft-secure-score-whats-coming.md)

Microsoft Güvenli Puanı, bu portalda https://security.microsoft.com/securescore [Microsoft 365 Defender bulunabilir](microsoft-365-defender.md#the-microsoft-365-defender-portal).

## <a name="july-2021"></a>Temmuz 2021

### <a name="added-improvement-action-related-to-microsoft-teams"></a>Çalışmayla ilgili geliştirme Microsoft Teams

- Arayarak bağlanılan kullanıcıların toplantı lobisini atlamalarını kısıtlama
- Dış katılımcıları toplantıda denetim sahibi Teams sınırlama
- Anonim kullanıcıların toplantı başlatmalarını Teams kısıtlama
- En son toplantılar için lobilerin Teams gerektirme
- Toplantılarda hangi kullanıcıların sunuma izin Teams yapılandırma

### <a name="added-improvement-action-related-to-microsoft-defender-for-endpoint"></a>Uç Nokta için Microsoft Defender ile ilgili geliştirme eylemi eklendi

- macOS için Uç nokta algılayıcısı veri toplama için Microsoft Defender'ı düzeltme
- macOS için Uç nokta engelli iletişimler için Microsoft Defender'ı düzeltme
- macOS'ta parola uzunluğunun en az 15 karakterine ayarlama
- macOS'ta 'Parola geçmişini zorunlu' olarak '24 veya daha fazla parola' olarak ayarlama
- macOS'ta 'En fazla parola yaşı' olarak '90 veya daha az gün' ayarlayın ancak 0'ı ayarlama
- macOS'ta hesap kilitleme eşiğini 5 veya daha düşük olarak ayarlama
- MacOS'ta Güvenlik Duvarı'nı açma
- Gatekeeper'i etkinleştir
- Sistem Bütünlüğü Korumasını (SIP) Etkinleştirme
- FileVault Disk Şifrelemesi'ne etkinleştirme
- MacOS'ta ekran koruyucu başlatıldığında ekranı kilit edecek şekilde ayarlama
- MacOS'ta ekran koruyucuların 20 dakika veya daha kısa bir sürede başlayacak şekilde ayarlanıyor olduğundan emin olmak
- Giriş Klasörlerinin Güvenliğini Sağlama
- macOS Microsoft Defender Virüsten Koruma gerçek zamanlı korumayı açma
- macOS Microsoft Defender Virüsten Koruma engelleme modunda PUA koruması açma
- macOS Microsoft Defender Virüsten Koruma bulut teslimi korumasını etkinleştirme
- macOS Microsoft Defender Virüsten Koruma tanımları güncelleştirme
- Linux için Uç nokta algılayıcısı veri toplaması için Microsoft Defender'ı düzeltin
- Linux için Uç nokta engelli iletişimler için Microsoft Defender'ı düzeltme
- Sınırsız Erişim Hesapları
- Linux Microsoft Defender Virüsten Koruma gerçek zamanlı korumayı açma
- Linux için Microsoft Defender Virüsten Koruma modunda MICROSOFT DEFENDER VIRÜSTEN KORUMA PUA korumasını açma
- Linux Microsoft Defender Virüsten Koruma bulut teslimi korumasını etkinleştirme
- Linux Microsoft Defender Virüsten Koruma tanımları güncelleştirme

Microsoft Teams, Güvenli Puan'da yeni bir geliştirme eylemi olarak "Anonim kullanıcıların toplantılara katılmalarını kısıtla" ifadesini görebilirler.

## <a name="december-2020"></a>Aralık 2020

### <a name="added-six-accounts-related-improvement-actions-for-microsoft-defender-for-endpoint"></a>Uç Nokta için Microsoft Defender'a altı hesapla ilgili geliştirme eylemi eklendi:

- 'En az parola uzunluğu' olarak '14 veya daha fazla karakter' ayarlayın
- 'Parola geçmişini zorunlu kılın' olarak '24 veya daha fazla parola' ayarlayın
- 'En fazla parola yaşı' olarak '60 veya daha az gün' ayarlayın ancak 0'a ayarlanmaz
- 'En az parola yaşı' olarak '1 veya daha fazla gün' ayarlayın
- Yerleşik Yönetici hesabını devre dışı bırakma
- Yerleşik Konuk hesabını devre dışı bırakma

## <a name="november-2020"></a>Kasım 2020

### <a name="removed-the-ability-to-create-servicenow-tickets-through-secure-score"></a>Güvenli Puan aracılığıyla ServiceNow bilet oluşturma özelliği kaldırıldı 

Güvenli Puan üzerinden ServiceNow bilet oluşturabilme özelliği > **ServiceNow** artık kullanılamaz. Sonraki adımları belirlerken, geri bildirimleriniz ve desteğiniz için teşekkür ederiz.

### <a name="added-three-services-related-improvement-actions-for-microsoft-defender-for-endpoint"></a>Uç Nokta için Microsoft Defender'a hizmetle ilgili üç geliştirme eylemi eklendi:

- E-Windows hizmetleri için Windows düzeltme
- Hizmet yürütülebilir yolu ortak korumalı bir konuma değiştirme
- Windows kayıt defterinde önbelleğe alınmış parolayı önlemek için hizmet hesabını değiştirme

## <a name="october-2020"></a>Ekim 2020

### <a name="removed-improvement-action-related-to-microsoft-defender-for-endpoint"></a>Uç nokta için Microsoft Defender ile ilgili geliştirme eylemi kaldırıldı

- Uyarı Microsoft Defender SmartScreen Windows Mağazası uygulaması web içeriği denetimi ayarlama

## <a name="august-2020"></a>Ağustos 2020

### <a name="updated-improvement-action-for-azure-active-directory"></a>Geliştirme eylemi güncelleştirildi Azure Active Directory

- Eski kimlik doğrulamayı engellemek için ilkeyi etkinleştirme

## <a name="incompatibility-with-identity-secure-score"></a>Kimlik Güvenliği Puanı ile uyumsuzluk

Microsoft Secure Score'ın son sürümüyle, geliştirilmiş bir puanlama modeli yayınlandı. Bu değişiklikler, güvenlik nedenlerinizi daha esnek ve doğru bir şekilde görüntülemenizi sağlar. Bununla birlikte, bu güncelleştirmeler Microsoft Güvenli Puanı'nın kimlik güvenliği puanıyla geçici olarak uyumsuz hale geldi.

Zaman içinde Identity Secure Score yeni puanlama modelini edinecek. Bu arada, müşteriler Microsoft Güvenli Puanı ve Kimlik Güvenli Puanı'nın rapor ettiği puanlarda farklılıklarını görebilir. Bu neden olan rahatsızlıktan dolayı özür dileriz. Bu deneyimlerin gelecekte daha uyumlu olduğundan emin olmak için çalışıyoruz.

- Kimlik geliştirme eylemleri için Microsoft Defender eklendi
- Uç Nokta Tehdit Güvenlik Açığı [Yönetimi &](/windows/security/threat-protection/microsoft-defender-atp/next-gen-threat-and-vuln-mgt) için Microsoft Defender Desteği
    - TVM tarafından sağlanan tüm yayın güvenlik önerileri artık kullanılabilir
