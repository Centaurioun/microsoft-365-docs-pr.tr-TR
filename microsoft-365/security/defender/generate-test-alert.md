# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Microsoft 365 Defender değerlendirme laboratuvarında test uyarısı oluşturma  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Aşağıdakiler için geçerlidir:**
- Microsoft 365 Defender

Microsoft 365 Defender değerlendirme laboratuvar kurulumu ve yapılandırmanızı tamamlamış olduğunuuza göre, tümleşik güvenlik çözümünün algılama, uyarı korelasyon ve kendi kendine düzeltme özelliklerini deneyimlemeniz için bir tehdit saldırı benzetimi çalıştırabilirsiniz.  

1. Oturum açmak için https://security.microsoft.com
2. Giriş sayfasında Benzetim başlıklı bir kutucuk **bulun**.  Microsoft 365 Defender'de test olayı oluşturma adımları için adım adım Microsoft 365 Defender.
