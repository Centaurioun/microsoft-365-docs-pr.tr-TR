# <a name="turn-on-the-preview-experience-in-microsoft-defender-for-endpoint"></a>Uç Nokta için Microsoft Defender'de önizleme deneyimini açma

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Şunlar için geçerlidir:**
- [Uç Nokta için Microsoft Defender Planı 1](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Uç Nokta için Microsoft Defender Planı 2](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


> Uç nokta için Defender'i deneyimlemek ister misiniz? [Ücretsiz deneme için kaydolun.](https://signup.microsoft.com/create-account/signup?products=7f379fee-c4f9-4278-b0a1-e4c8c2fcdf7e&ru=https://aka.ms/MDEp2OpenTrial?ocid=docs-wdatp-previewsettings-abovefoldlink)

Yaklaşan özellikleri ilk deneyenler arasında olmak için önizleme deneyimi ayarını açın.

1. Gezinti bölmesinde **Ayarlar** \> **Uç Noktaları** \> **Gelişmiş özellikler'i** seçin.

    :::image type="content" source="../../media/atp-preview-features-new.png" alt-text="Ayarlar ve önizleme deneyimi" lightbox="../../media/atp-preview-features-new.png":::


2. Ayarı **Açık** ve **Kapalı** arasında değiştirin ve **Tercihleri kaydet'i** seçin.
