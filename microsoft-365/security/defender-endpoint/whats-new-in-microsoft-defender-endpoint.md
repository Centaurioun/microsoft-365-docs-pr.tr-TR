## <a name="june-2021"></a>Haziran 2021

- [Delta dışarı aktarma yazılım açıkları değerlendirmesi](get-assessment-methods-properties.md#31-methods) API <br> Güvenlik açıkları ve [güvenli yapılandırma api'lerinin değerlendirmelerini dışarı aktarmaya ek](get-assessment-methods-properties.md) olarak. <br> Tam yazılım açıkları değerlendirmesinin (JSON yanıtı) aksine, cihaza göre kuruluş yazılım güvenlik açıkları değerlendirmesinin tüm anlık görüntüsünü elde etmek için kullanılır- değişiklikli dışarı aktarma API çağrısı yalnızca seçili tarihle geçerli tarih (değişiklik" API çağrısı) arasında olan değişiklikleri getirmek için kullanılır. Her zaman büyük miktarda veriyle tam dışarı aktarma yapmak yerine, yalnızca yeni, sabit ve güncelleştirilmiş güvenlik açıkları hakkında belirli bilgiler edinebilirsiniz. Delta dışarı aktarma API çağrısı, "kaç güvenlik açıkı düzeltilmiştir" veya "kuruluşa eklenen yeni güvenlik açıklarının kaç tane olduğu" gibi farklı KP'leri hesaplamak için de kullanılabilir.

- [Güvenlik açıklarının ve güvenli yapılandırmaların değerlendirmelerini dışarı aktarma](get-assessment-methods-properties.md) API <br> Her cihaz için ayrı veri Tehdit ve Güvenlik Açığı Yönetimi API koleksiyonu ekler. Farklı türde veriler almak için farklı API çağrıları vardır: güvenli yapılandırma değerlendirmesi, yazılım envanteri değerlendirme ve yazılım açıkları değerlendirmesi. Her API çağrısı, organizasyon cihazları için gerekli verileri içerir.

- [Düzeltme etkinliği](get-remediation-methods-properties.md) API <br> Kiracınız içinde oluşturulmuş düzeltme etkinlikleri içeren Tehdit ve Güvenlik Açığı Yönetimi API koleksiyonu ekler. Yanıt bilgileri kimlike göre tek bir düzeltme etkinliği, tüm düzeltme etkinlikleri ve tek bir düzeltme etkinliğinin açık cihazları içerir.

- [Cihaz bulma](device-discovery.md) <br> Ek cihazlara veya zahmetli süreç değişikliklerine gerek kalmadan şirket ağınıza bağlı, yönetimsiz cihazları bu şekilde bulmanızı sağlar. Yerleşik cihazları kullanarak ağ bağlantınız için yönelik olmayan cihazları bulabilir, güvenlik açıklarını ve riskleri değerlendirebilirsiniz. Daha sonra, ağ üzerinde yönelik olmayan uç noktalarla ilişkili riskleri azaltmak için bulunan cihazları eklemeye devam edin.

Mobil uygulama yönetimi hakkında daha fazla bilgi için bu [belgelere bakın](/microsoft-365/mem/intune/apps/mam-faq).

- [Microsoft Tunnel VPN tümleştirmesi](https://techcommunity.microsoft.com/t5/microsoft-defender-for-endpoint/announcing-new-capabilities-on-android-and-ios/ba-p/2442730) <br> Microsoft Tunnel VPN özellikleri artık Android için Uç Nokta uygulaması için Microsoft Defender ile tümleştirilmiştir. Bu sayede hem mobil tehdit savunması hem de mobil cihazlarından ön kaynaklara erişim olanağı sunarken, güvenlik ve IT ekipleri de tanıdıkları yönetici deneyimlerini sürdürebilme olanağına sahiptir.
