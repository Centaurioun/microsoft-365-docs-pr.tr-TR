# <a name="poland-drivers-license-number"></a>Polonya ehliyet numarası

## <a name="format"></a>Biçim

İki eğik çizgi içeren 11 veya 14 basamak

## <a name="pattern"></a>Desen

İki eğik çizgi içeren 11 veya 14 basamak

- beş basamak
- eğik çizgi
- iki basamak
- eğik çizgi
- dört veya yedi basamak
